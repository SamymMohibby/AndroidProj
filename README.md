# QuickTasks (Android)

Small Android app made for a course project.

## Features
- Add tasks with a text field + button
- Mark tasks done / undo
- Delete tasks
- Multiple views: Tasks, Stats, Settings
- Simple settings toggle (motivation text)

## Tech
- Android Studio
- Kotlin
- Jetpack Compose

## How to run
1. Open the project in Android Studio
2. Run on an emulator or Android device
3. Press "Run"

## Notes
Tasks are kept in memory only (not saved permanently).
